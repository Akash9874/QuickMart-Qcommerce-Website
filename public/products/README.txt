Product Images Directory

This directory should contain the following product images:
1. bread.jpg - Image of fresh bread
2. milk.jpg - Image of milk carton/bottle
3. eggs.jpg - Image of egg carton
4. bananas.jpg - Image of fresh bananas
5. apples.jpg - Image of fresh apples
6. pasta.jpg - Image of pasta package
7. juice.jpg - Image of orange juice
8. chicken.jpg - Image of organic chicken

Image Requirements:
- Format: JPG/JPEG
- Size: At least 800x800 pixels
- Aspect ratio: 1:1 (square)
- Max file size: 500KB

Note: These images should be high-quality product photos with white or transparent backgrounds.
Please ensure all images are properly licensed for commercial use.

Placeholder:
If a product image is missing, the application will display a default placeholder image.
You can customize the placeholder by adding a 'placeholder.jpg' file to this directory.
