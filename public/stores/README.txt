Store Logo Images for Quickmart

To add store logo images to your Q-commerce website, save image files with the following names in this directory:

1. freshmart.png - For the "FreshMart" store
2. quickstop.png - For the "QuickStop" store
3. valugrocer.png - For the "ValuGrocer" store

Logo requirements:
- Format: PNG with transparent background preferred
- Size: Recommended 200x200 pixels
- Style: Clean, simple logos that look good on both light and dark backgrounds
- File size: Keep under 100KB for faster loading

These logos will be displayed on the store comparison pages and in the partner stores section on the homepage.
